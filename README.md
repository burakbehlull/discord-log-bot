# Discord Log Bot

### Tech's:
Typescript, DiscordJS v14

#### Operating path: 
> npm i

> npm run dev

> npm run build

To enter bot settings, create an .env file, there is an example .env file named .env-example in the project:
```env
TOKEN = 
BOT_ID = 
logChannel = 
```
